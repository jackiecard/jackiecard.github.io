Botões

Os botões são usados com as classes do Bootstrap 3 (Ver tutorial completo).

Melhores Práticas de Uso

Botão comum:

<button class="btn-default">Nome do Botão</button>

Cancelar operações:

<button>Cancelar</button>

Botão Primário (cor oficial do TJSC):

<button class="btn-primary">Primário</button>

Botão de Sucesso:

<button class="btn-success">Sucesso</button>

Botão de Aviso:

<button class="btn-warning">Aviso</button>

Botão de Info:

<button class="btn-info">Info</button>

Botão de Perigo:

<button class="btn-danger">Perigo</button>

Desabilitados

Para desabilitar um botão basta colocar o atributo "disabled" dentro da tag "button".


<button disabled>Default</button>



Tipos de botões

Botão sem tipo

Botão clicável, sem nenhum elemento de ação.

<button class="btn-deafult">Nome do Botão</button>

Botão de tipo "button"

Usado para adicionar funções de scripts.

<button type="button" class="btn-deafult">Nome do Botão</button>

Botão de tipo "submit"

Envia todo o conteúdo do formulário, caso este possua uma "action".

<button type="submit" class="btn-deafult">Nome do Botão</button>

Botão de tipo "reset"

Apaga todo o conteúdo digitado nos campos, caso o usuário queira reescrevê-los.

<button type="reset" class="btn-deafult">Nome do Botão</button>
