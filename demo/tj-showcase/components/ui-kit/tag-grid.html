Grid System

Tags <grid> e <coluna>, representadas como:

Tamanho 3 Tamanho 6 Tamanho 3 Tamanho 2 Tamanho 5 Tamanho 5

Escrito como:

<grid>
	<coluna ocupa="3">Coluna 1</coluna>
	<coluna ocupa="6">Coluna 2</coluna>
	<coluna ocupa="3" nova-linha>Coluna 3</coluna>
	<coluna ocupa="2">Coluna 1</coluna>
	<coluna ocupa="5">Coluna 2</coluna>
	<coluna ocupa="5">Coluna 3</coluna>
</grid>

Renderizado como:

<div class="grid">
	<div class="col-3">Coluna 1</div>
	<div class="col-6">Coluna 2</div>
	<div class="col-3 nextline">Coluna 3</div>
	<div class="col-2">Coluna 1</div>
	<div class="col-5">Coluna 2</div>
	<div class="col-5">Coluna 3</div>
</div>
